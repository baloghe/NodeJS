Instructions

Three of You have to press the buttons in ascending order according to their labels. Pressing of a single button is considered as a subtask. Subtasks are assigned sequentially by the Server.
Each player has to push only the button the server assigns him. All buttons are active on Client side, however Server side listens only to the button of the Subtask just assigned.
Should the button be pushed by someone else, the task would be deemed as failed and reassigned.

Implementation: Chain of Promise on Sever side, socket.io for communication
